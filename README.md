# **Blackjack CLI**

A simple command-line Blackjack game written in C#.
Play against a computer dealer, hit or stand, and try to reach 21 without busting.

## Features

- Standard Blackjack rules

- Hit or Stand gameplay

- Automatic dealer logic

- Score calculation

- Ace value handling (1 or 11)

- Clear console UI

- Randomized card shuffling

- Win/Lose/Draw detection

## How to run

- Clone the repo:
  
  ``` git clone https://git.wwsi.edu.pl/DawidOldak/BlackjackCLI.git ```
- Navigate to the project folder:

  ``` cd BlackjackCLI ```
- Run the program:

  ``` dotnet run ```
